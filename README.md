# ✨ Lumi AI

**Lumi AI** is an artificial intelligence project created by **skizyzkxx**, designed to combine natural conversations with powerful development capabilities.

## 🚀 Features

### 💬 Chat Mode

* Natural and interactive conversations
* Answer questions and assist with daily tasks
* Simple and intuitive AI experience

### 🛠️ Build Mode

* Focused on programming and creation
* Generate complete projects from scratch
* Create games (like Snake 🐍), websites, and apps
* Supports multiple languages:

  * HTML
  * CSS
  * JavaScript
  * TypeScript
  * C#
* Automatically structures files for your project
* Allows customization and future expansion

## ⚠️ Current Status

> Lumi AI is currently under development.
> There are ongoing issues with the API integration, so some features may not work as expected at the moment.
